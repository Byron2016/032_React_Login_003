# Steps

- **Front End**

  - **Install and Setup Vite React**

    ```bash
      pnpm create vite
    ```

    - Name: auth-front
    - Project: React
    - TypeScript Option: TypeScript + SWC
    - Finish configuration

      - cd auth-front
      - pnpm install
      - pnpm run dev

    - **Add Routes Management**
      ```bash
        pnpm add react-router-dom
      ```
